#        PROYECTO G4 : Informalidad en Colombia
Repositorio en Stata que analiza la evolución de la probabilidad de empleo informal en Colombia durante la recuperación post-pandemia (2021–2025),
con el objetivo de caracterizar los cambios en el perfil de los trabajadores según nivel educativo y género.
